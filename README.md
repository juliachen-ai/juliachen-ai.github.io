# Julia Chen — Personal Website

A simple, single-page personal website for GitHub Pages.

## How to deploy on GitHub Pages (free hosting)

1. **Create a GitHub repo** — go to github.com/new and create a repo called e.g. `juliachen.github.io` (or any name)
2. **Add your headshot** — drop your updated headshot photo into this folder as `headshot.jpg`
3. **Push files** — upload `index.html`, `headshot.jpg`, and optionally `CNAME` to the repo
4. **Enable GitHub Pages** — go to repo Settings > Pages > set Source to "Deploy from a branch" > select `main` branch > Save
5. **Custom domain (optional)** — if you have a domain, add it in Settings > Pages > Custom domain, and create a `CNAME` file containing your domain

Your site will be live at `https://yourusername.github.io/reponame/` (or your custom domain).

## Files

- `index.html` — the full website (self-contained, no build step needed)
- `headshot.jpg` — your photo (you need to add this)
- `CNAME` — custom domain file (create if needed)
